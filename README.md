# AlphaPulse-Portfolio-Risk-Volatility-Monitor
📌 Project Overview

AlphaPulse is a financial analytics dashboard that monitors portfolio performance, volatility, and risk using real market data.

This project analyzes multiple stocks and applies risk metrics and simulation techniques to understand potential future portfolio outcomes.

🚀 Features

📈 30-Day Rolling Volatility

📊 Mean Daily Return Calculation

💰 Value at Risk (VaR 95%)

📉 Market Shock Analysis

🎲 Monte Carlo Simulation

🔍 Stock Return Relationship (Correlation Analysis)

📌 KPI Cards for quick insights

🛠️ Technologies Used

Python

Pandas

NumPy

Matplotlib / Seaborn

Financial Data Analysis Techniques

Monte Carlo Simulation

📊 Key Concepts Used
1️⃣ Rolling Volatility

Measures how much stock returns fluctuate over a 30-day window.

2️⃣ Mean Daily Return

Average daily percentage return of each asset.

3️⃣ Value at Risk (VaR 95%)

Estimates the maximum expected loss at a 95% confidence level.

4️⃣ Market Shock Simulation

Shows how portfolio value changes under negative market conditions.

5️⃣ Monte Carlo Simulation

Generates thousands of possible future portfolio values based on historical return distribution.
